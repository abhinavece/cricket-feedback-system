# Skills

Skills are specific capabilities with detailed instructions on HOW to perform tasks. Each skill contains:
- Description of the capability
- Step-by-step instructions
- Code patterns and examples
- Common pitfalls to avoid

## Available Skills

| Skill | File | Description |
|-------|------|-------------|
| Frontend Development | `frontend.md` | React, TypeScript, Tailwind, mobile-first |
| Backend Development | `backend.md` | Node.js, Express, API development |
| DevOps | `devops.md` | Docker, Kubernetes, Helm deployment |
| AI Service | `ai-service.md` | Python FastAPI, AI integration |
| Database | `database.md` | MongoDB, Mongoose, query optimization |
| Documentation | `documentation.md` | Feature documentation in CLAUDE.md and guides |

## Skill Activation

Skills are automatically activated based on:
1. File types being edited
2. Task description keywords
3. Manual reference in conversation

## Usage

Reference a skill by name:
- "Use the frontend skill to create a new component"
- "Apply the backend skill for this API endpoint"
- "Follow the devops skill for deployment"
